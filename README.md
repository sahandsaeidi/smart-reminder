# ⏰ Smart Reminder

A **smart reminder app** with **Notification**, **Service Worker**, and **PWA** support.  
You can install it on your phone like a native app!

---

## ✨ Features

- ✅ Add reminders with date & time
- 🔔 Get notifications at the right time
- 🔊 Sound alert when notification arrives
- 📱 Installable on mobile (PWA)
- 🌐 Persian & English languages
- 💾 Auto-save in browser (localStorage)
- 📋 List of active reminders


---



## 🛠️ Technologies

- HTML + CSS + JavaScript
- Service Worker (PWA)
- Notification API
- LocalStorage

---

## 📲 How to Install on Phone

1. Open the site in Chrome browser
2. Tap the **Install App** button
3. The app will be installed on your phone!
